ИТ-Учёт — инструкция по локальному запуску
============================================

ТРЕБОВАНИЯ:
  - Python 3.8 или новее (https://python.org)
  - При установке Python поставить галочку "Add Python to PATH"
  - Браузер (Chrome, Edge, Firefox)
  - Папка dist/ рядом с этим файлом (скачать билд из poehali.dev)

СТРУКТУРА ПАПОК:
  local_server/
    start.bat       ← запускать этот файл
    server.py
    README.txt
    it_uchet.db     ← создаётся автоматически (ваша база данных)
  dist/             ← билд сайта (скачать из poehali.dev → Скачать → Скачать билд)

КАК ЗАПУСТИТЬ:
  1. Скачайте билд: poehali.dev → Скачать → Скачать билд → распакуйте в папку dist/
  2. Дважды кликните start.bat
  3. Браузер откроется автоматически на http://localhost:8765

БАЗА ДАННЫХ:
  Все данные хранятся в файле it_uchet.db
  Это обычный SQLite файл — можно копировать как резервную копию.

ОСТАНОВКА:
  Нажмите Ctrl+C в окне сервера или просто закройте окно.
